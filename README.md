## ✨ Description

A **Currency Converter** CLI app written in Python that fetches real-time exchange rates from the internet and converts between any two currencies.

🌍 Supports **over 160 currencies** via [exchangerate.host](https://exchangerate.host).

🎯 **Features**:
- Convert between any two international currencies
- Live exchange rates using public API
- Simple command-line interaction

---

## 🛠️ How to Run

1. **Clone the repo**  
   ```bash
   git clone https://github.com/rafaelmoreirax/currency-converter.git
   cd currency-converter
   ```

2. **Install required packages**  
   This script requires the `requests` library. Install it via:
   ```bash
   pip install requests
   ```

3. **Run the script**  
   ```bash
   python currency_converter.py
   ```

4. **Follow the prompts**  
   Enter the amount, base currency, and target currency. Get the converted result instantly.
